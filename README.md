
## Name
Bootcamp OOP Camper

## Description
This Java console-based simulator models a camper's experience at a wilderness campsite. The core mechanics involve managing items, where players can bring and utilize gear with specific characteristics and abilities to aid their survival. Furthermore, the program simulates outdoor adventures, allowing the camper to explore distinct locations such as the Trail, River, and Mountain Spring, each designed with unique environmental dynamics.
